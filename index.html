What is CSS?

Cascading Style Sheets

CSS is the language used to style an HTML document. It describes how HTML elements should be displayed on screen, paper, or in other media.

  • Controls layout, colors, and fonts.
  • Saves a lot of work by controlling multiple web pages at once.
  • External stylesheets are stored in CSS files.

The Syntax

A CSS rule-set consists of a selector and a declaration block.

p {
  color: red;
  text-align: center;
}
  • Selector: Points to the HTML element (p).
  • Declaration: color: red;

CSS Selectors

Selectors are used to "find" (or select) HTML elements based on their element name, id, class, attribute, and more.

  • Element: div { }
  • ID: #myId { }
  • Class: .myClass { }
  • Universal: * { }

The Box Model

In CSS, the term "box model" is used when talking about design and layout.

  • Content: The text/image.
  • Padding: Clears area around content.
  • Border: A border around padding.
  • Margin: Clears area outside border.

Flexbox Layout

The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure.

.container {
  display: flex;
  justify-content: center;
}

Perfect for one-dimensional layouts (rows or columns).

CSS Grid Layout

Grid Layout is the most powerful layout system available in CSS. It is a 2-dimensional system.

  • Handles both columns and rows.
  • Complex "magazine-style" layouts.
  • Gap property to manage spacing.

CSS Animations

An animation lets an element gradually change from one style to another.

@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}

Change as many CSS properties as you want, as many times as you want.

Responsive Design

Making web pages look good on all devices.

Using the @media rule:

@media only screen and (max-width: 600px) {
  body { background-color: lightblue; }
}
