The Syntax

Rules to style elements.

The Box Model

Every element is a box.

  • Content: Actual data
  • Padding: Inner space
  • Margin: Outer space

Flexbox

One-dimensional layout system.

  • Justify-Content: Horizontal alignment
  • Align-Items: Vertical alignment
  • Items flex to fill space.

Animations

Make elements move without JavaScript.

@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
