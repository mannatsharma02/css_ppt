Today's Topic

CSS

Cascading Style Sheets

The language used to style an HTML document. It describes how HTML elements should be displayed on screen, paper, or in other media.

History & Evolution

CSS was first proposed by HÃ¥kon Wium Lie on October 10, 1994.

  • CSS1: Font properties, colors, background.
  • CSS2: Positioning, z-index, media types.
  • CSS3: Flexbox, Grid, Animations, 3D Transforms.

CSS Syntax

The set of rules that defines how to style elements.

p {
  color: red;
  text-align: center;
}
  • Selector: p (The HTML element)
  • Property: color (What to change)
  • Value: red (How to change it)

The Box Model

Every element in CSS is a box wrapped around the content.

(Look at the 3D model to the right/center)

  • Content: The actual text/image.
  • Padding: Space around content.
  • Border: A border around the padding.
  • Margin: Space outside the border.

Modern Layout: Flexbox

A one-dimensional layout method for laying out items in rows or columns.

  • Justify-Content: Aligns items horizontally.
  • Align-Items: Aligns items vertically.
  • Flex-Direction: Row or Column.

CSS Animations

CSS allows animation of HTML elements without using JavaScript or Flash!

@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}
